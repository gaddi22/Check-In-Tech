Installation Instructions
1. Install MySQL Community using the MySQL instructions in the Database folder
2. Move the server app’s jar and bat file to desired directory and execute the bat file. Use this bat whenever you wish to use the server
3. For remote check ins, move client app’s jar and bat file to a desired directory on a client machine.
	Note: The app will require the ip of the server app in order to perform check ins.
